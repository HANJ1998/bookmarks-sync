# 一键书签同步

将浏览器书签通过 WebDAV 同步至坚果云（或其他支持 WebDAV 的服务）。

## 功能

- ✅ **一键同步** — 点击弹窗即可将书签备份到云端
- ✅ **定时自动同步** — 支持每小时/6小时/12小时/每天
- ✅ **三种同步模式** — 本地优先 / 双向同步 / 云端优先
- ✅ **备份保留** — 自动清理过期备份文件
- ✅ **同步历史** — 查看最近 20 次同步记录

## 安装

### 开发模式加载

1. 打开 Chrome/Edge 浏览器，进入扩展管理页
2. 开启「开发者模式」
3. 点击「加载解压缩的扩展」
4. 选择 `src/` 目录

### 配置

1. 点击扩展图标 → 「高级设置」
2. 填入 WebDAV 地址、用户名、密码
3. 选择同步间隔和模式
4. 点击保存，扩展自动执行首次同步

## 兼容性

- Google Chrome （Manifest V3）
- Microsoft Edge （Manifest V3）

## 隐私

本扩展**不会**收集任何用户数据。所有书签数据仅保存在用户配置的 WebDAV 服务器上。
