# 一键书签同步

将浏览器书签通过 WebDAV 同步至私有云存储，支持自动定时备份与双向同步。

## 功能

- **一键同步** — 点击弹窗即可将书签备份到云端
- **定时自动同步** — 支持每小时、每 6 小时、每 12 小时、每天
- **三种同步模式** — 本地优先、双向同步、云端优先
- **备份保留** — 自动清理过期备份文件
- **同步历史** — 查看最近 20 次同步记录

## 安装

### 从 Release 安装

1. 前往 [Releases](https://github.com/HANJ1998/bookmarks-sync/releases) 下载最新版 zip
2. 解压到本地目录
3. 打开 Chrome/Edge → `chrome://extensions` / `edge://extensions`
4. 开启「开发者模式」
5. 点击「加载解压缩的扩展」→ 选择解压后的 `src/` 目录

### 配置

1. 点击扩展工具栏图标 → 「高级设置」
2. 填入 WebDAV 地址、用户名、密码
3. 选择同步间隔和模式
4. 点击保存

> WebDAV 地址示例：`https://dav.jianguoyun.com/dav/`（坚果云）

## 同步模式

| 模式 | 说明 |
|---|---|
| 以本地为准（推荐） | 将浏览器书签上传覆盖云端，忽略云端更改 |
| 双向同步 | 浏览器与云端双向合并，新增、修改、删除互相同步 |
| 以云端为准 | 用云端书签覆盖浏览器，忽略本地更改 |

## 兼容性

- Google Chrome（Manifest V3）
- Microsoft Edge（Manifest V3）

## 隐私

本扩展**不会**收集、存储或传输任何个人数据。

- 书签数据通过 WebDAV 协议直接传输到用户指定的服务器，**不经任何第三方**
- 配置信息存储在浏览器本地 `chrome.storage`
- 无第三方分析、广告或追踪服务
